→Key Technology Decisions
01
Java 21 Virtual Threads
One virtual thread per request, no thread pool tuning needed. I/O-bound throughput improves by 3-5x.
02
Seata Distributed Transactions
AT mode zero-invasion, TCC mode high performance, Saga mode long-running transactions. Choose the mode per scenario.
03
Redis Lua Atomic Operations
Stock deduction executes atomically in Redis single-threaded mode. No race conditions, no database locks. Hot path latency <2ms.
04
Canal Binlog Sync
Listens to MySQL/PostgreSQL binlog, syncs to Elasticsearch in real time. Zero invasion to business code.
05
Next.js 15 App Router
Server components reduce client-side JS size. Streaming rendering improves first-paint performance. Built-in i18n support.
