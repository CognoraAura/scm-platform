技术栈

每一层的技术选型都经过深思熟虑——不是追逐最新潮流，而是选择最适合问题的工具。

→关键选型决策
01
Java 21 虚拟线程
每个请求一个虚拟线程，无需线程池调优。I/O 密集型场景吞吐量提升 3-5 倍。
02
Seata 分布式事务
AT 模式零侵入，TCC 模式高性能，Saga 模式长事务。三种模式按场景选择。
03
Redis Lua 原子操作
库存扣减在 Redis 单线程中原子执行。无竞态条件，无数据库锁。热路径延迟 <2ms。
04
Canal binlog 同步
监听 MySQL/PostgreSQL binlog，实时同步到 Elasticsearch。业务代码零侵入。
05
Next.js 15 App Router
服务端组件减少客户端 JS 体积。流式渲染提升首屏性能。内置 i18n 支持。
